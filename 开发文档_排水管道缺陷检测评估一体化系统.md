# 排水管道缺陷检测与评估一体化系统 — 开发文档 v2

## 1. 项目概述

### 1.1 系统名称

排水管道缺陷智能检测与评估一体化系统（PipeAI）

### 1.2 系统定位

本系统是一个端到端的排水管道缺陷智能诊断平台。系统融合轻量化目标检测模型（YOLOv11n）与多模态大语言模型，通过"先检测、后评估"的两阶段协同架构，实现从视觉定位到语义判断的完整诊断链路。

系统提供两种工作模式：**问答模式**（交互式单图分析、标准知识查询、多轮对话）和**批量模式**（多图自动检测→评估→生成综合报告）。

### 1.3 核心设计理念

- **检测-评估协同**：检测模型负责"定位"，大模型负责"理解"，模拟人类专家先定位后判断的认知流程。
- **规程驱动**：将 CJJ 181-2012 标准从静态文本转化为可被大模型自主调用的工具，确保评估结论有据可依。
- **灵活模型接入**：同时支持云端多模态 API（通义千问、GPT-4o 等）和本地部署模型（vLLM），用户按需切换。
- **双模式工作流**：问答模式适合深入分析和交互讨论；批量模式适合工程场景下的大规模自动化处理。

### 1.4 技术栈总览

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| 前端 | React + Next.js + Tailwind CSS | 轻量级 SPA，支持对话/批量双模式 UI |
| 后端 | FastAPI (Python) | 提供 REST API + WebSocket 流式推送 |
| 智能体 | LangGraph | 基于有向状态图的 ReAct 推理框架 |
| 检测模型 | YOLOv11n（DINOv3 蒸馏后） | Ultralytics 推理，FP16 混合精度 |
| LLM 接入 | LiteLLM 统一适配层 | 统一对接云端 API 和本地 vLLM |
| 本地部署 | vLLM + LoRA 微调 | Qwen3-VL-8B，4-bit 量化加载 |
| 标准知识库 | JSON + 内存索引 | CJJ 181-2012 结构化存储 |
| 报告生成 | python-docx | Word 格式辅助评估报告 |

### 1.5 两种工作模式对比

| 维度 | 问答模式 | 批量模式 |
|------|---------|---------|
| 输入 | 单张图像 + 多轮文本对话 | 多张图像（文件夹/ZIP 上传） |
| 交互 | 实时对话、追问、质疑、查标准 | 一键启动，自动流水线执行 |
| 输出 | 对话式分析 + 可选单图报告 | 综合评估报告（含全部图像和评估详情） |
| LLM 调用 | 多轮 ReAct 推理 | 每张图像独立评估 + 最终汇总 |
| 适用场景 | 疑难缺陷分析、教学演示、标准学习 | 工程项目批量检测、日常巡检 |

```python
# backend/routers/chat.py — 问答模式路由

from fastapi import APIRouter, WebSocket, WebSocketDisconnect
from services.chat_engine import ChatEngine
from models.schemas import ChatMessage, ImageUploadResponse
import json, uuid, base64

router = APIRouter()

class ConnectionManager:
    """WebSocket 连接管理器，每个连接对应一个会话"""
    def __init__(self):
        self.connections: dict[str, WebSocket] = {}
        self.engines: dict[str, ChatEngine] = {}
    
    async def connect(self, websocket: WebSocket, session_id: str):
        await websocket.accept()
        self.connections[session_id] = websocket
        self.engines[session_id] = ChatEngine(session_id=session_id)
    
    def disconnect(self, session_id: str):
        self.connections.pop(session_id, None)
        self.engines.pop(session_id, None)

manager = ConnectionManager()

@router.websocket("/chat/ws")
async def chat_websocket(websocket: WebSocket):
    """
    问答模式 WebSocket 端点
    客户端发送: {"type": "text"|"image", "content": "...", "image_base64": "..."}
    服务端推送: {"type": "token"|"tool_call"|"result"|"error", "data": "..."}
    """
    session_id = str(uuid.uuid4())[:8]
    await manager.connect(websocket, session_id)
    
    # 发送会话ID给客户端
    await websocket.send_json({"type": "session", "session_id": session_id})
    
    try:
        while True:
            data = await websocket.receive_json()
            engine = manager.engines[session_id]
            
            if data["type"] == "text":
                # 纯文本消息（对话/查询标准）
                async for event in engine.handle_text_message(data["content"]):
                    await websocket.send_json(event)
            
            elif data["type"] == "image":
                # 图像上传 + 可选文本指令
                image_bytes = base64.b64decode(data["image_base64"])
                text = data.get("content", "请评估图像中的缺陷")
                async for event in engine.handle_image_message(image_bytes, text):
                    await websocket.send_json(event)
            
            elif data["type"] == "detect_only":
                # 仅执行检测（不调用LLM）
                image_bytes = base64.b64decode(data["image_base64"])
                result = engine.run_detection_only(image_bytes)
                await websocket.send_json({"type": "detection_result", "data": result})
    
    except WebSocketDisconnect:
        manager.disconnect(session_id)
```

```python
from fastapi import APIRouter, UploadFile, File, BackgroundTasks
from typing import List
from services.batch_engine import BatchEngine
from models.schemas import BatchTask, BatchProgress

# 批量任务状态存储（生产环境应替换为 Redis）
batch_tasks: dict[str, BatchTask] = {}

@router.post("/batch/start")
async def start_batch(
    files: List[UploadFile] = File(...),
    background_tasks: BackgroundTasks = None,
    llm_provider: str = "cloud",  # "cloud" 或 "local"
):
    """
    启动批量处理任务
    接收多张图像，后台执行: 检测 → 评估 → 生成综合报告
    """
    task_id = str(uuid.uuid4())[:12]
    
    # 保存上传的图像
    import tempfile, os
    task_dir = os.path.join("uploads/batch", task_id)
    os.makedirs(task_dir, exist_ok=True)
    
    image_paths = []
    for f in files:
        path = os.path.join(task_dir, f.filename)
        with open(path, "wb") as out:
            out.write(await f.read())
        image_paths.append(path)
    
    task = BatchTask(
        task_id=task_id,
        status="pending",
        total_images=len(image_paths),
        processed=0,
        image_paths=image_paths,
        results=[],
        report_path=None,
    )
    batch_tasks[task_id] = task
    
    # 启动后台处理
    if background_tasks:
        background_tasks.add_task(run_batch_pipeline, task_id, llm_provider)
    
    return {"task_id": task_id, "total_images": len(image_paths)}

@router.get("/batch/{task_id}/status")
async def get_batch_status(task_id: str):
    """查询批量任务进度"""
    task = batch_tasks.get(task_id)
    if not task:
        return {"error": "Task not found"}
    return {
        "task_id": task_id,
        "status": task.status,
        "total": task.total_images,
        "processed": task.processed,
        "current_image": task.current_image,
        "errors": task.errors,
    }

@router.get("/batch/{task_id}/report")
async def download_batch_report(task_id: str):
    """下载批量任务的综合报告"""

        return {"error": "Report not ready"}
```

## 4. LLM Provider 统一适配层

### 4.1 设计思路

使用 LiteLLM 作为统一适配层，通过相同的 OpenAI 兼容接口对接不同供应商的模型。无论底层是通义千问云端 API、GPT-4o，还是本地 vLLM 部署，上层 LangGraph Agent 的代码完全不变。

### 4.2 实现

```python
# backend/services/llm_provider.py

from openai import OpenAI
from typing import Optional, AsyncIterator
import base64

class LLMProvider:
    """
    统一 LLM 接入层
    支持: 通义千问VL系列 / GPT-4o / 本地vLLM
    所有模型通过 OpenAI 兼容 API 调用，上层代码无感知
    """
    
    # 预设配置
    PRESETS = {
        "qwen-vl-max": {
            "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
            "supports_vision": True,
            "max_context": 32000,
        },
        "qwen-vl-plus": {
            "base_url": "https://dashscope.aliyuncs.com/compatible-mode/v1",
            "supports_vision": True,
            "max_context": 32000,
        },
        "gpt-4o": {
            "base_url": "https://api.openai.com/v1",
            "supports_vision": True,
            "max_context": 128000,
        },
        "qwen3-vl-8b-local": {
            "base_url": "http://localhost:8000/v1",
            "supports_vision": True,
            "max_context": 8192,
        },
    }
    
    _instance = None
    _config = None
    
    def __init__(self, config: dict = None):
        if config is None:
            config = self._config or {
                "model_name": "qwen-vl-max",
                "base_url": self.PRESETS["qwen-vl-max"]["base_url"],
                "api_key": "",
                "temperature": 0.1,
                "max_tokens": 4096,
            }
        
        self.model_name = config["model_name"]
        self.base_url = config.get("base_url", self.PRESETS.get(self.model_name, {}).get("base_url", ""))
        self.api_key = config.get("api_key", "")
        self.temperature = config.get("temperature", 0.1)
        self.max_tokens = config.get("max_tokens", 4096)
        
        self.client = OpenAI(
            api_key=self.api_key or "not-needed",
            base_url=self.base_url,
        )
    
    @classmethod
    def update_global_config(cls, config):
        """更新全局配置，所有新建的 LLMProvider 实例使用新配置"""
        cls._config = {
            "model_name": config.model_name,
            "base_url": config.base_url,
            "api_key": config.api_key,
            "temperature": config.temperature,
            "max_tokens": config.max_tokens,
        }
    
    def chat_with_image(self, system_prompt: str, user_text: str, 
                        image_path: str = None, image_base64: str = None) -> str:
        """
        发送带图像的多模态请求
        
        Args:
            system_prompt: 系统提示词
            user_text: 用户文本
            image_path: 图像文件路径（与 image_base64 二选一）
            image_base64: base64编码的图像
        """
        messages = [{"role": "system", "content": system_prompt}]
        
        user_content = []
        if image_path or image_base64:
            if image_path:
                with open(image_path, "rb") as f:
                    image_base64 = base64.b64encode(f.read()).decode()
            user_content.append({
                "type": "image_url",
                "image_url": {"url": f"data:image/jpeg;base64,{image_base64}"}
            })
        user_content.append({"type": "text", "text": user_text})
        
        messages.append({"role": "user", "content": user_content})
        
        response = self.client.chat.completions.create(
            model=self.model_name,
            messages=messages,
            max_tokens=self.max_tokens,
            temperature=self.temperature,
        )
        return response.choices[0].message.content
    
    def chat_text_only(self, system_prompt: str, messages: list) -> str:
        """纯文本对话（用于标准查询等不需要图像的场景）"""
        full_messages = [{"role": "system", "content": system_prompt}] + messages
        response = self.client.chat.completions.create(
            model=self.model_name,
            messages=full_messages,
            max_tokens=self.max_tokens,
            temperature=self.temperature,
        )
        return response.choices[0].message.content
    
    def get_langchain_llm(self):
        """获取 LangChain 兼容的 LLM 实例（供 LangGraph 使用）"""
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(
            model=self.model_name,
            api_key=self.api_key or "not-needed",
            base_url=self.base_url,
            temperature=self.temperature,
            max_tokens=self.max_tokens,
        )
```

```python
# backend/services/detector.py

from ultralytics import YOLO
from PIL import Image
import io, json

class PipelineDefectDetector:
    """排水管道缺陷检测器"""
    
    CLASS_MAP = {
        0: {"code": "PL", "name": "破裂"},
        1: {"code": "CK", "name": "错口"},
        2: {"code": "SG", "name": "树根"},
        3: {"code": "SL", "name": "渗漏"},
        4: {"code": "TL", "name": "脱落"},   # CJJ181全称: 接口材料脱落
        5: {"code": "ZW", "name": "障碍物"},
    }
    
    CONFIDENCE_THRESHOLD = 0.5  # 检测层风控阈值
    
    def __init__(self, model_path: str, device: str = "cuda:0"):
        self.model = YOLO(model_path)
        self.device = device
    
    def detect_from_bytes(self, image_bytes: bytes, img_size: int = 640) -> dict:
        """从图像字节数据执行检测"""
        img = Image.open(io.BytesIO(image_bytes))
        return self._run_detect(img, img_size)
    
    def detect_from_path(self, image_path: str, img_size: int = 640) -> dict:
        """从文件路径执行检测"""
        img = Image.open(image_path)
        return self._run_detect(img, img_size)
    
    def detect_batch(self, image_paths: list, img_size: int = 640) -> list:
        """批量检测"""
        images = [Image.open(p) for p in image_paths]
        results_list = self.model.predict(
            source=[img for img in images],
            imgsz=img_size,
            conf=self.CONFIDENCE_THRESHOLD,
            device=self.device,
            half=True,
            verbose=False,
        )
        return [self._parse_single(r, img) for r, img in zip(results_list, images)]
    
    def _run_detect(self, img: Image.Image, img_size: int) -> dict:
        results = self.model.predict(
            source=img,
            imgsz=img_size,
            conf=self.CONFIDENCE_THRESHOLD,
            device=self.device,
            half=True,
            verbose=False,
        )
        return self._parse_single(results[0], img)
    
    def _parse_single(self, result, img: Image.Image) -> dict:
        orig_w, orig_h = img.size
        detections = []
        
        if result.boxes is not None:
            for box in result.boxes:
                cls_id = int(box.cls[0])
                conf = float(box.conf[0])
                xywhn = box.xywhn[0].tolist()
                xyxy = box.xyxy[0].tolist()
                cls_info = self.CLASS_MAP.get(cls_id, {"code": "UNK", "name": "未知"})
                
                detections.append({
                    "class_code": cls_info["code"],
                    "class_name": cls_info["name"],
                    "bbox": {"x_center": round(xywhn[0],4), "y_center": round(xywhn[1],4),
                             "width": round(xywhn[2],4), "height": round(xywhn[3],4)},
                    "bbox_pixels": {"x1": int(xyxy[0]), "y1": int(xyxy[1]),
                                    "x2": int(xyxy[2]), "y2": int(xyxy[3])},
                    "confidence": round(conf, 4),
                    "position_description": self._describe_position(xywhn),
                })
        
        detections.sort(key=lambda d: d["confidence"], reverse=True)
        return {
            "detections": detections,
            "image_size": {"width": orig_w, "height": orig_h},
            "total_defects": len(detections),
        }
    
    @staticmethod
    def _describe_position(xywhn):
        xc, yc = xywhn[0], xywhn[1]
        h = "左侧" if xc < 0.33 else ("右侧" if xc > 0.67 else "中部")
        v = "上部" if yc < 0.33 else ("下部" if yc > 0.67 else "中部")
        return f"图像{v}{h}区域"
    
    def close(self):
        del self.model
```

```python
class StructuredPriorConverter:

        lines = []
        if high_conf:

                )
        
        if low_conf:
```

```python
# backend/services/chat_engine.py

from langgraph.graph import StateGraph, END
from langchain_core.messages import (SystemMessage, HumanMessage, AIMessage, 
                                      ToolMessage, BaseMessage)
from typing import TypedDict, Annotated, Sequence
import operator, time, base64, io, tempfile, os

class ChatState(TypedDict):
    messages: Annotated[Sequence[BaseMessage], operator.add]
    session_id: str
    detection_cache: dict        # 当前会话的检测结果缓存
    uploaded_image_path: str     # 当前会话上传的图像路径

class ChatEngine:
    """
    问答模式引擎 — 基于 LangGraph 的 ReAct 智能体
    支持: 多轮对话 / 图像上传 / 标准查询 / 缺陷评估 / 报告生成
    """
    
    def __init__(self, session_id: str):
        self.session_id = session_id
        self.image_path = None
        self.detection_cache = {}
        self.graph = self._build_graph()
    
    def _build_graph(self):
        """构建 LangGraph 状态图"""
        from services.tools import get_all_tools
        from services.prompts import SYSTEM_PROMPT
        from services.llm_provider import LLMProvider
        
        llm = LLMProvider().get_langchain_llm()
        tools = get_all_tools()
        tool_map = {t.name: t for t in tools}
        llm_with_tools = llm.bind_tools([t.get_schema() for t in tools])
        
        # --- 节点定义 ---
        
        def model_reasoning(state: ChatState) -> dict:
            """模型推理节点：调用LLM决策"""
            messages = list(state["messages"])
            if not any(isinstance(m, SystemMessage) for m in messages):
                messages.insert(0, SystemMessage(content=SYSTEM_PROMPT))
            response = llm_with_tools.invoke(messages)
            return {"messages": [response]}
        
        def tool_execution(state: ChatState) -> dict:
            """工具执行节点"""
            last = state["messages"][-1]
            tool_messages = []
            for tc in last.tool_calls:
                tool = tool_map.get(tc["name"])
                try:
                    validation = tool.validate_params(tc["args"]) if tool else {"valid": False, "error": f"Unknown tool: {tc['name']}"}
                    if not validation["valid"]:
                        result = f"参数校验失败：{validation['error']}。请修正参数后重试。"
                    else:
                        result = tool.execute(**tc["args"])
                except Exception as e:
                    result = f"工具执行异常：{str(e)}"
                tool_messages.append(ToolMessage(content=str(result), tool_call_id=tc["id"]))
            return {"messages": tool_messages}
        
        def should_continue(state: ChatState) -> str:
            last = state["messages"][-1]
            if hasattr(last, "tool_calls") and last.tool_calls:
                return "call_tool"
            return "finish"
        
        # --- 构建状态图 ---
        graph = StateGraph(ChatState)
        graph.add_node("model_reasoning", model_reasoning)
        graph.add_node("tool_execution", tool_execution)
        graph.set_entry_point("model_reasoning")
        graph.add_conditional_edges("model_reasoning", should_continue, 
                                    {"call_tool": "tool_execution", "finish": END})
        graph.add_edge("tool_execution", "model_reasoning")
        
        return graph.compile()
    
    async def handle_text_message(self, text: str):
        """处理纯文本消息（对话/查标准）"""
        initial_state = {
            "messages": [HumanMessage(content=text)],
            "session_id": self.session_id,
            "detection_cache": self.detection_cache,
            "uploaded_image_path": self.image_path,
        }
        
        async for event in self.graph.astream(initial_state, stream_mode="updates"):
            for node_name, node_output in event.items():
                if node_name == "model_reasoning":
                    msg = node_output["messages"][-1]
                    if hasattr(msg, "tool_calls") and msg.tool_calls:
                        for tc in msg.tool_calls:
                            yield {"type": "tool_call", "data": {"tool": tc["name"], "args": tc["args"]}}
                    else:
                        yield {"type": "token", "data": msg.content}
                elif node_name == "tool_execution":
                    for msg in node_output["messages"]:
                        yield {"type": "tool_result", "data": msg.content[:500]}
    
    async def handle_image_message(self, image_bytes: bytes, text: str):
        """处理图像+文本消息"""
        from services.detector import PipelineDefectDetector
        from services.prior_converter import StructuredPriorConverter
        
        # 保存图像到临时文件
        tmp_path = os.path.join("uploads", f"{self.session_id}.jpg")
        os.makedirs("uploads", exist_ok=True)
        with open(tmp_path, "wb") as f:
            f.write(image_bytes)
        self.image_path = tmp_path
        
        # 执行检测
        detector = PipelineDefectDetector(model_path="models/detector/best.pt")
        detection_result = detector.detect_from_bytes(image_bytes)
        self.detection_cache = detection_result
        
        # 转换先验
        converter = StructuredPriorConverter()
        prior_text = converter.convert(detection_result)
        
        # 构建包含先验的用户消息
        full_message = f"{text}\n\n## 检测先验信息\n{prior_text}"
        
        initial_state = {
            "messages": [HumanMessage(content=full_message)],
            "session_id": self.session_id,
            "detection_cache": detection_result,
            "uploaded_image_path": tmp_path,
        }
        
        # 先推送检测结果
        yield {"type": "detection_result", "data": detection_result}
        
        # 然后流式推送评估过程
        async for event in self.graph.astream(initial_state, stream_mode="updates"):
            for node_name, node_output in event.items():
                if node_name == "model_reasoning":
                    msg = node_output["messages"][-1]
                    if hasattr(msg, "tool_calls") and msg.tool_calls:
                        for tc in msg.tool_calls:
                            yield {"type": "tool_call", "data": {"tool": tc["name"], "args": tc["args"]}}
                    else:
                        yield {"type": "token", "data": msg.content}
    
    def run_detection_only(self, image_bytes: bytes) -> dict:
        """仅执行检测（不调用LLM）"""
        from services.detector import PipelineDefectDetector
        detector = PipelineDefectDetector(model_path="models/detector/best.pt")
        return detector.detect_from_bytes(image_bytes)
```

```python
        if image_path and os.path.exists(image_path):
            doc.add_heading("检测图像", level=1)
            doc.add_picture(image_path, width=Inches(5.5))
        
        self._write_defect_sections(doc, results)
        self._write_disclaimer(doc)
        
        os.makedirs(os.path.dirname(output_path) or ".", exist_ok=True)
        doc.save(output_path)
    
    def _write_defect_sections(self, doc, results):
        summary = results.get("summary", {})
        doc.add_heading("一、缺陷概况", level=1)
        doc.add_paragraph(
            f"本次评估共检测到 {summary.get('total_count', 0)} 处缺陷，"

        )
        
        doc.add_heading("二、逐项评估详情", level=1)
        grade_colors = {1: RGBColor(0,128,0), 2: RGBColor(255,165,0),
                       3: RGBColor(255,128,0), 4: RGBColor(255,0,0)}
        
        for i, defect in enumerate(results.get("defects", []), 1):
            doc.add_heading(f"缺陷 {i}：{defect.get('name','')} ({defect.get('code','')})", level=2)
            doc.add_paragraph(f"缺陷描述：{defect.get('description','')}")
            gp = doc.add_paragraph()
            gr = gp.add_run(f"等级判定：{defect.get('grade','?')}级")
            gr.bold = True
            gr.font.color.rgb = grade_colors.get(defect.get('grade'), RGBColor(0,0,0))
            doc.add_paragraph(f"判定依据：{defect.get('criteria','')}")
            doc.add_paragraph(f"特征分析：{defect.get('analysis','')}")
        
        doc.add_heading("三、置信度说明", level=1)
        for d in results.get("defects", []):
            doc.add_paragraph(f"{d.get('name','')}缺陷评估置信度：{d.get('confidence_level','中')}",
                            style='List Bullet')
        
        doc.add_heading("四、修复建议与优先级", level=1)
        for d in sorted(results.get("defects",[]), key=lambda x: x.get("grade",0), reverse=True):
            doc.add_paragraph(f"[{d.get('grade','?')}级] {d.get('name','')}：{d.get('repair_suggestion','')}",
                            style='List Bullet')
    
    def _write_disclaimer(self, doc):

        p = doc.add_paragraph()

class BatchReportGenerator:
    """
    批量模式综合报告生成器
    包含: 所有图像 + 每张图像的缺陷检测结果 + 评估详情 + 汇总统计
    注意: 这不是CJJ 181-2012附录D的标准检测成果表，而是辅助评估参考报告
    """
    
    def generate(self, all_assessments: list, output_path: str):
        doc = Document()
        
        # ===== 封面 =====
        title = doc.add_heading("排水管道缺陷批量评估报告", level=0)
        title.alignment = WD_ALIGN_PARAGRAPH.CENTER
        
        meta = doc.add_paragraph()
        meta.add_run(f"生成时间：{datetime.datetime.now().strftime('%Y-%m-%d %H:%M:%S')}\n")
        meta.add_run(f"评估图像数量：{len(all_assessments)} 张\n")
        meta.add_run("评估标准：CJJ 181-2012《城镇排水管道检测与评估技术规程》\n")
        meta.add_run("评估方式：AI辅助批量评估（辅助决策参考）\n")
        
        # ===== 汇总统计 =====
        doc.add_heading("总体统计", level=1)
        self._write_summary_table(doc, all_assessments)
        
        # ===== 逐图详情 =====
        doc.add_heading("逐图评估详情", level=1)
        for i, assessment in enumerate(all_assessments, 1):
            self._write_single_image_section(doc, i, assessment)
        
        # ===== 声明 =====
        self._write_disclaimer(doc)
        
        os.makedirs(os.path.dirname(output_path) or ".", exist_ok=True)
        doc.save(output_path)
    
    def _write_summary_table(self, doc, all_assessments):
        """写入汇总统计表"""
        total_defects = 0
        grade_dist = {1: 0, 2: 0, 3: 0, 4: 0}
        type_dist = {}
        error_count = 0
        
        for a in all_assessments:
            if a["status"] == "error":
                error_count += 1
                continue
            for d in a.get("detection", {}).get("detections", []):
                total_defects += 1
                code = d["class_code"]
                type_dist[code] = type_dist.get(code, 0) + 1
        
        doc.add_paragraph(f"总处理图像数：{len(all_assessments)} 张")
        doc.add_paragraph(f"检测到的缺陷总数：{total_defects} 处")
        doc.add_paragraph(f"处理失败数：{error_count} 张")
        
        # 缺陷类型分布表
        if type_dist:
            doc.add_heading("缺陷类型分布", level=2)
            table = doc.add_table(rows=1, cols=3)
            table.style = 'Table Grid'
            hdr = table.rows[0].cells
            hdr[0].text, hdr[1].text, hdr[2].text = "缺陷类型", "代码", "数量"
            name_map = {"PL":"破裂","CK":"错口","SG":"树根","SL":"渗漏","TL":"脱落","ZW":"障碍物"}
            for code, count in sorted(type_dist.items(), key=lambda x: -x[1]):
                row = table.add_row().cells
                row[0].text = name_map.get(code, code)
                row[1].text = code
                row[2].text = str(count)
    
    def _write_single_image_section(self, doc, index, assessment):
        """写入单张图像的评估详情"""
        doc.add_heading(f"图像 {index}：{assessment['image_name']}", level=2)
        
        # 嵌入图像
        img_path = assessment["image_path"]
        if os.path.exists(img_path):
            doc.add_picture(img_path, width=Inches(4.5))
        
        # 检测结果摘要
        detection = assessment.get("detection", {})
        defects = detection.get("detections", [])
        doc.add_paragraph(f"检测到 {len(defects)} 处缺陷")
        
        if defects:
            dt = doc.add_table(rows=1, cols=4)
            dt.style = 'Table Grid'
            h = dt.rows[0].cells
            h[0].text, h[1].text, h[2].text, h[3].text = "缺陷类型", "置信度", "位置", "坐标"
            for d in defects:
                r = dt.add_row().cells
                r[0].text = d["class_name"]
                r[1].text = f"{d['confidence']*100:.1f}%"
                r[2].text = d["position_description"]
                b = d["bbox_pixels"]
                r[3].text = f"({b['x1']},{b['y1']})-({b['x2']},{b['y2']})"
        
        # AI评估结果
        if assessment["status"] == "success" and assessment.get("assessment_text"):
            doc.add_heading("AI评估结果", level=3)
            # 将Markdown格式的评估文本按段落写入
            for para_text in assessment["assessment_text"].split("\n\n"):
                if para_text.strip():
                    doc.add_paragraph(para_text.strip())
        elif assessment["status"] == "error":
            doc.add_paragraph(f"评估失败：{assessment.get('error', '未知错误')}")
        
        # 分页
        doc.add_page_break()
    
    def _write_disclaimer(self, doc):
        doc.add_paragraph()
        p = doc.add_paragraph()
        p.add_run("声明：").bold = True
        p.add_run(
            "本报告由AI系统批量自动生成，定位为面向辅助决策的可解释性评估意见。"
            "本报告不是CJJ 181-2012标准附录D中定义的检测成果表，"
            "不具备工程归档效力。建议由持证技术人员逐条审核确认后使用。"
        )
```

```typescript
// app/chat/page.tsx — 问答模式核心页面

"use client";
import { useState, useRef, useEffect } from "react";
import useWebSocket from "react-use-websocket";
import ChatPanel from "@/components/chat/ChatPanel";
import ImageUploader from "@/components/chat/ImageUploader";
import DetectionOverlay from "@/components/chat/DetectionOverlay";
import ModelSelector from "@/components/common/ModelSelector";

export default function ChatPage() {
  const [messages, setMessages] = useState<Message[]>([]);
  const [detectionResult, setDetectionResult] = useState<DetectionResult | null>(null);
  const [uploadedImage, setUploadedImage] = useState<string | null>(null);
  const [isProcessing, setIsProcessing] = useState(false);
  
  const { sendJsonMessage, lastJsonMessage } = useWebSocket(
    `ws://${process.env.NEXT_PUBLIC_API_HOST}/api/chat/ws`,
    {
      onOpen: () => console.log("WebSocket connected"),
      shouldReconnect: () => true,
    }
  );
  
  // 处理服务端推送
  useEffect(() => {
    if (!lastJsonMessage) return;
    const msg = lastJsonMessage as any;
    
    switch (msg.type) {
      case "session":
        console.log("Session:", msg.session_id);
        break;
      case "detection_result":
        setDetectionResult(msg.data);
        break;
      case "token":
        // 流式追加 AI 回复
        appendToLastMessage(msg.data);
        break;
      case "tool_call":
        addSystemMessage(`🔧 调用工具: ${msg.data.tool}`);
        break;
      case "tool_result":
        addSystemMessage(`📋 工具返回结果`);
        break;
    }
  }, [lastJsonMessage]);
  
  const handleSend = (text: string) => {
    setMessages(prev => [...prev, { role: "user", content: text }]);
    sendJsonMessage({ type: "text", content: text });
  };
  
  const handleImageUpload = async (file: File, prompt: string) => {
    const base64 = await fileToBase64(file);
    setUploadedImage(URL.createObjectURL(file));
    setDetectionResult(null);
    sendJsonMessage({ type: "image", image_base64: base64, content: prompt });
  };
  
  return (
    <div className="flex h-[calc(100vh-4rem)]">
      {/* 左侧: 图像预览 + 检测结果 */}
      <div className="w-1/3 border-r p-4 overflow-y-auto">
        <ImageUploader onUpload={handleImageUpload} />

      {/* 右侧: 对话区域 */}

        <ChatPanel 
          messages={messages} 
          onSend={handleSend}
          isLoading={isProcessing}
        />
      </div>
    </div>
  );
}
```

```typescript
// app/batch/page.tsx — 批量模式核心页面

"use client";
import { useState } from "react";
import BatchUploader from "@/components/batch/BatchUploader";
import ProgressBar from "@/components/batch/ProgressBar";
import BatchResultTable from "@/components/batch/BatchResultTable";

export default function BatchPage() {
  const [taskId, setTaskId] = useState<string | null>(null);
  const [progress, setProgress] = useState<BatchProgress | null>(null);
  const [results, setResults] = useState<BatchResult[]>([]);
  
  const startBatch = async (files: FileList) => {
    const formData = new FormData();
    Array.from(files).forEach(f => formData.append("files", f));
    formData.append("llm_provider", "cloud");
    
    const res = await fetch("/api/batch/start", { method: "POST", body: formData });
    const data = await res.json();
    setTaskId(data.task_id);
    
    // 轮询进度
    const interval = setInterval(async () => {
      const statusRes = await fetch(`/api/batch/${data.task_id}/status`);
      const status = await statusRes.json();
      setProgress(status);
      
      if (status.status === "completed") {
        clearInterval(interval);
      }
    }, 2000);
  };
  
  return (
    <div className="max-w-5xl mx-auto p-6 space-y-6">
      <BatchUploader onStart={startBatch} disabled={!!taskId && progress?.status !== "completed"} />
      
      {progress && (
        <>
          <ProgressBar 
            current={progress.processed} 
            total={progress.total}
            currentImage={progress.current_image}
            status={progress.status}
          />
          <BatchResultTable results={results} />
        </>
      )}
      
      {progress?.status === "completed" && (
        <a 
          href={`/api/batch/${taskId}/report`}
          className="btn btn-primary"
          download
        >
          📄 下载综合评估报告(Word)
        </a>
      )}
    </div>
  );
}
```

## 16. 风险控制体系

### 16.1 三级风控

| 级别 | 位置 | 机制 |
|------|------|------|
| 第一级 | 检测模块输出 | confidence < 0.5 → 过滤 |
| 第二级 | 评估模块推理 | confidence < 0.85 → 审慎评估，提示不确定性 |
| 第三级 | 系统输出端 | AI 置信度低 → 标注 + 建议人工复核 |

### 16.2 数据安全

- 云端模式：图像经 base64 编码传输至 API，需评估供应商的数据安全政策
- 本地模式：全部数据不出本机，适合敏感工程数据
- 前端提供一键切换，用户可根据数据敏感度选择模式

### 16.3 报告定位声明

所有报告（单图/批量）均需包含免责声明：本报告是 AI 辅助的**辅助决策参考**，不是 CJJ 181-2012 附录 D 定义的检测成果表，不具备工程归档效力。
